Spotlight: Masters Thesis

My Masters thesis, entitled "Leveraging Grammers For OpenMP Development in Supercomputing Environments", motivates and describes my thesis project, a website that provides a step by step flow to run code on a supercomputer. Each step aims to reduce the barrier to entry by attaching a graphical user interface and instructions to repeat the steps without Spur.
